# 后续功能与体验改进记录

记录时间：2026-05-01

## 当前已完成

- 快速搜索增强：已支持中文、拼音、首字母、英文模糊匹配、命中高亮和匹配类型标记。
- Popup 同步状态面板：已显示状态、数据来源、上次同步、自动同步、状态消息，并支持立即同步。
- 飞书配置向导：已完成 3 步式配置、凭证填写、连接测试与字段检查。
- 链接管理轻量体验版：已支持重复 URL 检测、favicon 自动建议、图标预览和更明确的表单校验。
- 基础测试：已添加 `node --test`，覆盖快速搜索核心、Popup 状态格式化核心、飞书配置检查核心和链接管理纯逻辑。
- 最近提交：`41342e1 增强快速搜索并添加同步状态面板`。

## 未完成事项

### 1. 链接管理体验升级后续

当前状态：已支持添加、编辑、删除、图标 URL、同分类拖拽排序、重复 URL 检测、favicon 建议和图标预览。

可做方向：
- 批量导入/导出导航数据，支持 JSON 备份与恢复。
- 从浏览器书签导入，需要新增 `bookmarks` 权限并提供权限说明。
- 支持导入冲突处理：跳过、覆盖、重命名或按分类合并。
- 添加导入预览，保存前显示将新增、跳过和冲突的链接数量。

推荐优先级：高。

### 2. 分类能力补齐

当前状态：拖拽排序仅支持同分类内排序。

可做方向：
- 跨分类拖拽移动网站。
- 分类重命名。
- 分类排序。
- 空分类管理。

推荐优先级：中高。

### 3. 搜索进阶能力

当前状态：已完成拼音、首字母和模糊搜索，但搜索仍只负责打开网站。

可做方向：
- 最近访问优先。
- 收藏或置顶网站。
- 搜索结果按分类分组。
- 命令面板化：在搜索里直接执行“添加网站”“打开设置”“切换分类”等命令。

推荐优先级：中。

### 4. 背景设置实时预览

当前状态：设置页能配置背景图、遮罩和模糊，但需要保存后回新标签页查看效果。

可做方向：
- 在设置页增加背景预览区域。
- 调整图片 URL、上传图片、遮罩和模糊时实时更新预览。
- 保存前不影响新标签页当前背景。

推荐优先级：中。

### 5. 首次使用和空状态优化

当前状态：未配置飞书或暂无链接时，提示较弱。

可做方向：
- 新用户第一屏提供三条路径：继续用测试模式、配置飞书、导入书签。
- 空分类/空搜索/无缓存时使用更明确的行动按钮。
- 将“测试模式”和“飞书模式”的差异直接显示在界面里。

推荐优先级：中。

### 6. 设置页安全和易用性

当前状态：`APP_SECRET` 是密码输入框，但没有显示/隐藏、脱敏或字段辅助。

可做方向：
- `APP_SECRET` 显示/隐藏按钮。
- 敏感字段脱敏展示。
- 字段说明和复制按钮。
- 保存前进行本地格式校验。

推荐优先级：中。

## 推荐下一步

建议下一步优先做“链接管理导入/导出与书签导入”。

理由：
- 轻量链接管理体验已经覆盖日常新增和编辑，下一步应解决迁移、备份和批量维护问题。
- 书签导入能显著降低从浏览器原生书签迁移到导航页的成本。
- 这部分需要新增权限和冲突处理规则，适合单独设计和测试。
