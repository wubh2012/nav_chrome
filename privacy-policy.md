# 隐私政策 - 水果导航

## 概述

水果导航（"本扩展程序"）是一款基于飞书多维表格的 Chrome 浏览器起始页扩展。我们高度重视您的隐私和个人信息保护。本隐私政策说明了我们如何收集、使用、存储和保护您的信息。

使用本扩展即表示您同意本隐私政策所述的做法。如果您不同意，请立即停止使用本扩展程序。

## 信息收集

本扩展程序可能收集以下类型的信息：

### 1. 个人身份信息

- **飞书账户信息：**当您配置飞书多维表格同步功能时，我们会访问您提供的飞书应用凭证（App ID、App Secret、App Token、Table ID）。这些信息仅用于与飞书 API 进行身份验证和数据同步，不会存储在我们的服务器上。
- **导航数据：**您通过飞书多维表格管理网站书签、分类和排序信息。这些数据存储在您的飞书多维表格中，我们仅在您授权的情况下读取和同步。

### 2. 使用数据

- **本地存储：**我们使用 Chrome 扩展存储 API（chrome.storage.local）将以下数据保存在您的本地设备上：
  - 导航数据缓存（便于离线访问）
  - 主题偏好设置
  - 同步状态和配置
  - 飞书访问令牌（已加密存储，有效期 2 小时）
- **使用统计：**我们可能会收集匿名化的使用统计数据（如功能使用频率、错误报告等），用于改进产品体验。所有统计数据均不包含个人身份信息。

### 3. 技术信息

- **设备信息：**Chrome 浏览器版本、操作系统类型等基本技术信息。
- **使用日志：**扩展程序运行时的基本日志（仅用于故障排查）。

## 信息使用

我们收集的信息仅用于以下目的：

- 提供和维护导航起始页功能
- 与飞书多维表格同步您的导航数据
- 保存您的主题偏好设置
- 改进产品功能和用户体验
- 排查技术问题和错误
- 防止欺诈和滥用

> **重要声明：**我们不会出售、出租或以其他方式将您的个人信息提供给第三方，除非获得您的明确同意或法律要求。

## 数据存储与安全

- **本地存储：**您的导航数据和配置信息存储在 Chrome 浏览器的本地存储中，受 Chrome 安全机制保护。
- **飞书数据：**您的导航书签和分类信息存储在您自己的飞书多维表格中，受飞书隐私政策保护。
- **数据安全：**我们采用行业标准的安全措施保护您的数据，包括加密传输（HTTPS）和安全存储实践。
- **数据保留：**本地缓存数据保留 7 天，之后自动过期。您可以随时在扩展设置中清除所有本地数据。

## 第三方服务

本扩展程序可能使用以下第三方服务：

- **飞书开放平台：**用于数据同步和存储。数据将传输至飞书服务器，受飞书服务条款和隐私政策约束。
- **Google Favicon 服务：**用于获取网站图标。访问的域名信息将发送至 Google 服务器。

## 您的权利

您享有以下权利：

- **访问权：**您可以随时通过扩展设置查看和管理您的配置。
- **删除权：**您可以清除所有本地存储的数据，或从飞书多维表格中删除导航数据。
- **导出权：**您可以导出配置备份（未来版本将提供此功能）。
- **控制权：**您可以随时禁用或卸载本扩展程序，这将清除所有本地存储的数据。

## 数据清除

您可以通过以下方式清除数据：

- **清除本地缓存：**在扩展设置页面点击"清除缓存"按钮
- **重置所有设置：**在扩展设置页面点击"重置所有"按钮
- **卸载扩展：**卸载本扩展将自动清除所有本地存储数据
- **清除飞书数据：**直接从飞书多维表格中删除记录

## 隐私政策更新

我们可能会不时更新本隐私政策。重大变更将通过以下方式通知：

- 在 Chrome 扩展更新说明中公告
- 在扩展设置页面显示通知

我们建议您定期查看本隐私政策。发布后继续使用本扩展即表示您接受更新后的政策。
